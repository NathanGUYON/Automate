To access the application, first unzip the application archive. The project is also accessible via this link : https://github.com/NathanGUYON/Automate .

To start the application, you must open the project with Intellij IDEA then launch the "Monapplication" program (the green "Run" button).

To access the application's Javadoc, you can access it via the link : https://nathanguyon.github.io/Automate/Logiciels/Javadoc/tp1/package-summary.html .

If you want to access it via files, go to the "Javadoc" folder then click on the "index" file which will bring you to the summary.

If the app displays an error on startup, which can happen, restart it and the issue should be resolved.
